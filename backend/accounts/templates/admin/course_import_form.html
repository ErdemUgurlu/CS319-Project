Instructions

Upload an Excel file (.xlsx or .xls) containing course information to bulk import courses and sections.

Your Excel file must include the following columns:

Department, Course Code, Course Title, Credits, Section Count, Student Count

Note:

  • Department: Department code (e.g., CS, EE, MATH)
  • Course Code: Course code without department prefix (e.g., 101, 202)
  • Course Title: Full name of the course
  • Credits: Numeric credit value (can include decimals)
  • Section Count: Number of sections to create (integer)
  • Student Count: Number of students per section (integer)
